# portfolio
このポートフォリオは、React.jsを用いて構築されたWebアプリケーションです。

## 概要
- **主な機能:**
  - ログイン機能
  - ログアウト機能
  - ページ遷移

- **動作確認:**
  - URL：レバテックマイページに記載
  - 稼働時間：月～金 08:00～16:00
    ※状況に応じて、システムの起動を見合わせる可能性があります。
  
## リポジトリ構成
- pnpmを活用したmonorepo構成です。
  - フロントエンド：app_frontend + shared
  - バックエンド  ：app_api      + shared
